# README Image Assets

Add the following exported dashboard images to this folder:

| Filename | Suggested Content |
| --- | --- |
| `executive-dashboard.png` | Executive KPIs, segment net gain, and portfolio opportunity |
| `churn-explainability.png` | Customer churn score, risk drivers, segment, and intervention |
| `retention-experiments.png` | Control/treatment uplift, ROI, and deployment decision |
| `customer-segments.png` | Segment comparison and risk-versus-value view |
| `project-architecture.png` | Final project architecture diagram |

Use PNG files with a consistent browser width. A 16:9 or wide desktop crop works well for GitHub.
